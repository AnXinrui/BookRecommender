#### 快速运行：

1. 克隆本仓库并下载

2. 创建虚拟环境并且进入项目目录

```bash
conda create -n env_name python=3.11 -y
```

```bash
conda activate env_name
```

3. 安装需要的包

```bash
pip install -r requirements.txt
```

4. 运行该项目

```bash
streamlit run app.py
```
